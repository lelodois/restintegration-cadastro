# Rest Integração

### Preço - Cadastro  
Projeto com informações cadastrais da produto filial  
Run PrecoCadastroApplication.java http://localhost:9083/swagger-ui.html  

## Arquitetura

* RestAPI
* Java 8
* Maven
* Spring boot / Mvc / Event
* Tomcat embedded
* Hibernate
* Swagger
* H2
* Fluent Http
